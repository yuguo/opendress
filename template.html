模板规范

/**
    *主页模板
**/
QZONE.template = xExtend(QZONE.templateBase, {
	//数据声明
	data : "",
	//模板声明
	template : "",
	//脚本声明
	js:"",
	//启动脚本
	bootstrap:function(){}
});

数据声明

data 用来声明该套装需要加载的数据类型，不声明则默认不加载该类型数据，缺省时默认为加载所有类型数据。比如：

data : "photos,posts"
//声明调用了相册和说说的数据

不同类型间用,隔开，类型与对应关系见下表：

数据类型 声明字段
日志 blogs
说说 posts
相册 posts
个人档 profile
留言板 messages
访客 visitor

模板声明

template 用来定义该套装的HTML结构，包含HTML标签、标签属性和模板数据。定义的模板结构将会被插入到页面模板的导航条和copyright之间。

HTML标签：请使用W3C标准的HTML标签，当然也可以使用HTML5的新标签，但是所有标签必须闭合，并且标签字母为小写。

标签属性：请在开始标签中定义标签的属性，标签属性名称为小写，属性的值由双引号 " 包围。我们推荐你使用accesskey和tabindex来提升页面的易用性。

模板变量：Qzone开放装扮使用doT.js作为模板引擎，所有的模板数据均由前台模板吐出。在使用之前请先了解doT.js语法。模板变量可与HTML混合在一起，可以作为结点的文本内容输出，也可以作为节点的属性值。

以下代码定义了一个日志列表的模板结构：

<ul class='fn_blogs'>
    {{~it.blogs :value:index}}
    <li class='blog_item'>
        <a href='{{=value.link}}'>{{=value.title}}</a>
    </li>
    {{~}}
</ul>

脚本声明

为了实现某些动态交互的效果，你可以使用 jQuery 作为脚本框架，引用脚本的方法很简单，只需在脚本声明中指定要调用的脚本的名称:

js:"jQuery"
