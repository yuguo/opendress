日志（blogs）

属性和方法 名称 返回值 备注
data_source 日志标识 字符串 blogs
title 日志标题 文本
link 日志连接 url
images 获取日志里的图片 图片对象数组
summary 摘要 文字
post_time 发表时间 日期时间 参考时间日期格式化
category 分类 文本
visit_count 阅读数 数字
comment_count 评论数 数字

说说（posts）

属性和方法 名称 返回值 备注
data_source 数据类型标识 字符串 posts
type 说说类型 字符串 original (原创),retweet (转发),photo (带图说说)
content() 说说内容 HTML片段 总是返回一般情况下需要展现的信息，如果转发帖会有转发链
original_content() 说说内容 HTML片段 仅是包含当前用户发表的信息
broadcast_content2() 说说内容 HTML片段 针对于转发帖的字段，会带有被转发的原文信息
post_time 发表时间 日期时间 参考时间日期格式化
comment_count 评论数 数字
medias() 媒体数组 数组 数组中的元素为媒体，参考媒体
